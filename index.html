Shop logic

This tool uses a fixed-panel model rather than a removable picture frame model, which matches your build method.

Inside capturesign + 2 × recess
Outside sizeinside capture + 2 × frame width
Auto frame widthbased on max sign dimension

For an 8 × 10 sign, a 1.25 in face width looks balanced and is a sensible starting point for a shop sign frame.

Standard guidance

A strict standard recess by frame size is not really the norm; 0.25 in per side is simply a common practical default in framing guidance and works well for small engraved sign panels.

For this fixed assembly style, you can keep 0.25 in as the default and only change it when you want a more delicate edge reveal or more aggressive sign capture.
